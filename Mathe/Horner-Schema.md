Das Horner Schema ist eine Anwendungsmöglichkeit um Nullstellen zu berechnen. bei einer
[[n-funktion]] dritten grades
 Hier führ muss man zuerst eine Nullstelle Erraten. wie zb. $X_{1}= 3$ 
Dann wird Das Schema angewendet.
Beispiel 
$f(x)=0=x^{3}-2x^{2}-5x+6$

|            | 1   | -2  | -5  | 6   |
| ---------- | --- | --- | --- | --- |
| $X_{01}=3$ | 0   |     |     |     |
|            |     |     |     |     |
Hierbei werden die Zahlen die Übereinanderstehen voneinander Addiert.
Das Ergebniss davon wird mit der Eratenen X stelle Multipliziert und in die Nächste Reihe gesteckt| 

|            | 1   | -2  | -5  | 6   |
| ---------- | --- | --- | --- | --- |
| $X_{01}=3$ | 0   | 3   | 3   | -6  |
|            | 1   | 1   | -2  | 0   |
Bei der Letzten Addition muss dabei immer Null rauskommen.
Das Ergebniss lässt sich so aufstellen:
$$
1x^{2}+1x-2
$$
